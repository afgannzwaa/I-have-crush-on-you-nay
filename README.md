# I-have-crush-on-you-nay

# 🌸 Project For Naya

Web interaktif estetik dengan tema **Aurora Glassmorphism** untuk menyampaikan pesan spesial secara personal.

## ✨ Fitur Utama
* **Smooth Animation:** Transisi 60 FPS menggunakan GSAP.
* **Typing Effect:** Efek ngetik otomatis yang dramatis.
* **Smart PNG:** Gambar transparan menyatu sempurna tanpa kotak putih.
* **Data Integrated:** Jawaban tersimpan otomatis via Formspree.
* **Mobile Ready:** Responsif di semua ukuran layar HP.

## 🛠️ Setup Cepat
1. **Assets:** Siapkan `love1.mp3` dan `gif1.gif` sampai `gif7.gif`.
2. **Formspree:** Ganti `URL` di JavaScript dengan ID Formspree milikmu.
3. **Run:** Buka `index.html` di browser.

## 📦 Tech Stack
* **UI:** HTML, CSS (Glassmorphism), GSAP.
* **Logic:** JavaScript, Typed.js, SweetAlert2.
* **Backend:** Formspree API.
